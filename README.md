# Dictionary-Bookmark

Basic idea:
The Smart Dictionary Bookmark is a compact electronic bookmark that allows users to browse and search word definitions 
while reading. Designed as a portable learning tool, it combines hardware and software to make vocabulary building more convenient, 
interactive, and accessible. Its lightweight, bookmark-sized design allows readers to quickly access definitions without 
interrupting their reading experience.

How to use it: You simply use it by pressing buttons, to type out words on. matrix keyboard. AFter that simply view the 
definition of the word on your screen.

Why: As a bookworm, I am always reading across words that I don't know the definition of. I figured that a device that you could 
kep with you while reading, anad has a built-in dictionary, migth just be the perfect tool to pull out.
