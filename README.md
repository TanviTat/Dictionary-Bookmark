# Dictionary-Bookmark

Basic idea:
The Smart Dictionary Bookmark is a compact electronic bookmark that allows users to browse and search word definitions 
while reading. Designed as a portable learning tool, it combines hardware and software to make vocabulary building more convenient, 
interactive, and accessible. Its lightweight, bookmark-sized design allows readers to quickly access definitions without 
interrupting their reading experience.

How to use it: You simply use it by pressing buttons, to type out words on. matrix keyboard. AFter that simply view the 
definition of the word on your screen.

Why: As a bookworm, I am always reading across words that I don't know the definition of. I figured that a device that you could 
kep with you while reading, anad has a built-in dictionary, migth just be the perfect tool to pull out.

Justification for each part:
Raspberry Pi Pico 2- The Raspberry Pi Pico 2 is the main controller of the project. It will run the code, handle user input, 
and control the display and other components.
0.96" OLED Display- The OLED display will show words, definitions, menus, and other information for the user.
4x4 Matrix Keypad- The keypad will let users navigate the device and enter letters when searching for words.
Breadboard- The breadboard will be used to test the circuit before making permanent connections. It makes it easy to change 
and improve the design during development.
Jumper Wires- Jumper wires will connect the different components together while testing on the breadboard.
USB Cable- The USB cable will be used to upload code to the Raspberry Pi Pico 2 and provide power during testing.
Switch- The switch will allow the user to turn the device on and off easily.
3.7V LiPo Battery- The LiPo battery will power the bookmark so it can be used without being plugged into a computer.
LiPo Charging Board- The charging board will safely recharge the battery and make the device reusable without replacing batteries.
Small Magnets- The magnets will help keep the bookmark closed and secure when it is not being used.
Tiny Hinges- The hinges will allow parts of the bookmark case to open and close while protecting the electronics inside.
Protoboard- The protoboard will be used to build a more permanent version of the circuit after testing is complete.
Soldering Kit- The soldering kit will be used to permanently connect components and wires when assembling the final prototype.
